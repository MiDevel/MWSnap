General settings

Warn at start about conflicting hotkeys

Check this option if you want to be notified about conflicting system-wide hotkeys. In case of hotkeys conflicts go to the Hotkeys configuration dialog and select unique hotkeys for items marked in red.

Auto-start with Windows

Check this option if you want MWSnap to start automatically with Windows to be always available. When enabling this option, consider checking also the "Start minimized" option.

Minimize to tray

Check this option if you want minimized MWSnap to appear as a tiny icon in the system tray (where the clock is displayed) instead of a standard big button in the task bar.

Note that left-clicking MWSnap's icon in the system tray restores the program window. Right-clicking it opens a popup menu with all most often used commands.

Start minimized

Check this option if you want the program to start in minimized mode. Note that it's possible to start in minimized mode only when minimizing to the system tray has been selected.

Minimize on closing the main window using the button [X]

Check this option if you want the program to minimize despite the system Close button has been clicked. Regardless of this setting selecting the Exit button will always terminate the program.

Always show icon in tray

Check this option if you want the program to show an icon in the system tray even when the application is not minimized. It's recommended to have this option activated because the tray icon offers a quick access to the capturing menu (invoked with right-clicking the icon.)

Confirm files overwriting

Check this option if you want the program to warn you if a file with the same name you are trying to use for saving already exists.

Undo stack size

Use this field to specify the number of image states MWSnap should retain for undoing. Set the number to 0 to disable Undo.
Warning!
For large images each Undo step can consume even 1-2 MB of RAM.
