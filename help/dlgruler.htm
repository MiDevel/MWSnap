Ruler tool

The ruler tool is a comfortable tool for quick measuring of sizes of screen objects (in pixels.) After invoking the command a ruler similar to the one shown below appears:

The ruler can be freely dragged over the screen using the mouse simply by dragging any of its points. To resize the ruler by 5 pixels use the arrows keys (with <Shift> to resize precisely, and with <Ctrl> to resize by 50 pixels). To quickly resize the ruler drag the widget in the bottom-right corner. To close the ruler tool and to return back to the program press <Esc> or select "Close" from the local menu. Note that the number to the right of the "hole" shows the precise distance of the mouse cursor position to the "0" on the scale.

The ruler's "hole" contains a hidden functionality. When clicked, it pops up a local menu containing the following options:

Horizontal ruler

Enables the horizontal ruler (default). Shortcut: <Ctrl+H>.

Vertical ruler

Enables the vertical ruler. Shortcut: <Ctrl+V>.

Toggle the ruler direction

Changes the direction of the scale on the ruler from left-to-right to right-to-left. The arrows next to the ruler "hole" show the current direction. It's also possible to click the arrows to quickly change the direction. Shortcut: <Ctrl+D>.

Help

Shows this help page.

Close

Closes the ruler tool. Shortcuts: <Ctrl+F4> or <Esc>.
