Menu "Tools"

Ruler tool

Use this command to activate the ruler tool. The ruler tool can be used for measuring (in pixels) sizes of screen elements.

Zoom tool

Use this command to activate the zoom tool. The zoom tool can be used for magnifying selected parts of the screen.

Color picker tool

Use this command to activate the color picker tool. The color picker tool can be used for reading the color information of selected pixels on the screen.

Window info tool

Use this command to activate the window info tool. The window info tool can be used to read various parameters of opened windows or their controls.

Hotkeys

Use this command to activate/deactivate and redefine MWSnap's system-wide hotkeys. System-wide hotkeys are active even if MWSnap's window has no focus, or is minimized.

Auto-saving

Use this command to activate/deactivate automatic saving of all captured images. Use the "Settings" command to define saving parameters. Current state of auto-saving is displayed in the tool tip of the "Auto-saving" button of the toolbar.

Auto-printing

Use this command to activate/deactivate automatic printing of all captured images. Images will be printed using last parameters specified manually. Current state of auto-saving is displayed in the tool tip of the "Auto-printing" button of the toolbar.

Language

Use this command to select one of available language translations of the program. The selected language will be instantly activated and used in future sessions with the program.

Settings

Use this command to open a dialog box allowing configuring MWSnap.
