Menu "View"

Zoom in

Use this command to zoom in (magnify) the view of the image. The current zoom level is displayed on the status bar. Keyboard shortcut: <+>.

Zoom reset

Use this command to reset zoom - set 100%.

Zoom out

Use this command to zoom out the view of the image. The current zoom level is displayed on the status bar. Keyboard shortcut: <->.

Picture toolbar

Use this command to show/hide the picture toolbar shown below the preview panel.

Capture panel

Use this command to show/hide the preview panel.

Status bar

Use this command to show/hide the status bar.

Show all

Use this command to show all user interface elements.

Show none

Use this command to show all user interface elements.
