Status bar

Status bar is displayed at the very bottom of the application's window. It's separated into three panels:

  • info panel,
  • zoom level panel,
  • image size panel.

Info panel shows various information during working with the program. It shows more detailed descriptions of focused user interface elements like buttons, and is used for displaying information about the program state and its last actions, like auto-saving.

Zoom level panel shows the current zoom level as a percent of the magnification. 100% means no zooming. Minimal zoom is 10%, maximal 2000%.

Image size panel displays the size of the currently captured image. Note that if the size does not appear on the fixed sizes list, the "Add fixed size" command is active.
