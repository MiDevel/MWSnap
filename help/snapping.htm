Performing snaps

The main function of MWSnap is performing screen captures that can be next saved, auto-saved, printed, or put on the clipboard. Current version of MWSnap offers the following capturing modes:

  • fixed-size rectangle,
  • any area,
  • a window, a menu or a control,
  • full desktop,
  • repeat last capture.

Capturing can be invoked using the program's menu, the toolbar, the capture panel, or system-wide hot keys.
