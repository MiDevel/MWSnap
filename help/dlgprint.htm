Printing

The printing dialog allows to specify several parameters for printing the current captured image.

Position

Select the proper option to define the location of the image on the sheet of paper.

Size

The controls allow selecting one of three available scaling modes.

"Original" - no scaling is performed, the image will be printed "as is".

"Scale" - enter the scaling factor to be applied to the image before printing. Values greater than 1 will magnify the printout. Values in range <0..1> will reduce it.

"Fit to page" - the image will be magnified/scaled down so that it it occupies maximum space on the printable area of the paper. Note that the image will keep its proportion.

"Smooth scaling" - check this option if you want the image to be scaled using one of available rendering methods what usually produces smoother printouts.

Page orientation

Check the desired paper orientation.

Header, footer

Check the checkboxes to enable printing a header and/or a footer on the paper. Long text fields allow to define the contents of the header and of the footer. The following templates can be used in the definition:

%d - print date (short)
%D - print date (long)
%t - print time (short)
%T - print time (long)
%x - picture width (in pixels)
%y - picture height (in pixels)
%n - file name
%p - file path

Any other text will appear as entered.
