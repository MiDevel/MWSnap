Sounds settings

Sound enabled

Check this option if you want to enable sounds in the program.

List of events

Use the list of events to activate the event for which you would like to configure the sound. Current version allows for customizing  the following events:

  • Confirmations
  • Errors and warnings
  • Snapping done
  • Saving done
  • Image copied to the clipboard
  • Image pasted from the clipboard

Play built-in sound

Select this option if you want the program to use the built-in sound for the active event. Remember to click on the "Save" button to save this change.

Play this .wav file:

Select this option if you want the program to use the user wave file for the active event. Remember to click on the "Save" button to save this change.

This sound is active

Check this option if you want to enable the sound for the active event. Note that the "Sound enabled" setting can be used to disable sound for all events without changing this setting individually for every item. Remember to click on the "Save" button to save this change.

Test

Select this button to hear the currently selected sound effect. If you are satisfied, click on the "Save" button to save this state.

Save

Select this button to save the sound settings of the active event.
