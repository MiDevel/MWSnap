Menu "File"

Open

Use this command to display a standard Windows dialog which allows you to open an existing image file of any supported format. Such opened image can be next saved in a selected format or transformed using any of the program editing options.

Note that image files can be also opened by simply dragging them from any file explorer and dropping onto the program's window, or by using the integrated picture viewer.

Save as...

Use this command to save the current captured image in any location, using any file name and picture format. The program remembers last used saving location and suggests it automatically. The command is not active if no image has been captured yet.
Note that captured images can be also auto-saved.

Print...

Use this command to print the current captured image to the default Windows printer. After selecting the command a dialog window appears where the user can select printing parameters and define optional headers and footers.

Exit

Use this command to terminate the application.
